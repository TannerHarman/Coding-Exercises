What is the difference between single, double, and backtick quotes for strings?
    - `Backticks` are used for String interpolation
    - "Double Quotes" can be used when you need to single quote something inside of a string
    - 'Single Quotes' are the most basic way to define a string. You'll need to use escape characters to add single quotes properly.
    
Which type of quote lets you embed variables/expressions into a string?
    - Backticks
    
How do you embed variables/expressions into a string?
    - You use backticks and add you variable with the following syntax `${variable}`
    
How do you escape characters in a string?
    - You use a \ in front of the character you need to escape.

What are the three logical operators and what do they stand for?
    - || - This means 'Or'. So if one of the two things being compared is true, then the result is true[]
    - && - this means 'And'. Both things being compared need to be true for the result to be true.
    - ! - This means 'Not'. This returns true if the things being compared don't match or arent' equal.

    - > - Greater than symbol
    - <= - Less than or Equal to
    - >= - Greater than ore Equal to
    - == - Equals - Checks to see if the values are equal
    - === - Strict Equals - Checks to see if both the values & data Types are equal.
